# Cypress Cucumber (BDD) Practice

This repository contains my **Cypress + Cucumber (BDD)** practices and assignments completed during my QA automation training.

> **Note:**  
> This project setup was cloned from an instructor’s repository. 
> All test scenarios, implementations, and exercises in this repository were completed by me as hands-on practice.

## **Technologies Used**
- **Cypress**
- **Cucumber (BDD)**
- **JavaScript**
- **Page Object Model (POM)**

## **Project Structure**

```text
cypress
 ├─ e2e
 │   ├─ features
 │   └─ step_definitions
 ├─ pages
 └─ support
features → Gherkin scenarios written in .feature files
step_definitions → Step implementations
pages → Page Object Model structure
support → Cypress custom commands and configuration
```
## **Implemented Scenarios**

### **Login Practice**
- Basic login flow implemented using **Cypress + Cucumber**
- Includes:
  - `.feature` file written in **Gherkin**
  - Step Definitions
  - Page Object Model structure

This practice focuses on:
- Gherkin syntax usage  
- Step-to-code mapping  
- Clean and reusable POM design
